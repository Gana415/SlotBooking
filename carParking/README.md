CAR PARKING
==================
The project has run in 2 formats ...

# Run the file Format
* Pass the file name to arguments `args[0]`
* Run as Java Application get the response in console.
* Go to Run -> Debug Configurations set the arguments path file as given    below.
![High Level Diagram](filePathArgument.png)

## Run the Commend Format
* Run as Java Application get the response in console.
* Given input as format of below options.<br /><br />
	 `create_parking_lot <number>`<br />
	 `park <registerNumber> <colour>`<br />
	 `leave <number>`<br />
	 `status <>`<br />
	 `registration_numbers_for_cars_with_colour <colour>`<br />
	 `slot_number_for_registration_number <registrationNumber>`<br />
	 `slot_numbers_for_cars_with_colour <colour>`<br />
	 `exit <>`<br />
